# 🚖 Uber Power BI Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze Uber trip data and provide actionable business insights through interactive visualizations and KPIs.

The dashboard helps stakeholders monitor ride performance, customer behavior, revenue trends, trip patterns, and operational efficiency using real-world business intelligence techniques.

---

## 🎯 Objectives

- Analyze Uber trip performance
- Monitor booking trends
- Identify revenue patterns
- Understand customer behavior
- Track trip distance and duration
- Evaluate ride categories
- Create executive-level KPI dashboards

---
